# NL Design

**NL Design** is a Nextcloud app that applies [NL Design System](https://nldesignsystem.nl/) design tokens to your Nextcloud instance. It provides consistent, government-compliant theming for Dutch municipalities, provinces, and national organizations.

## What it does

- Applies **39 organization-specific themes** (Rijkshuisstijl, Amsterdam, Utrecht, Den Haag, Rotterdam, and 34 more)
- Maps NL Design System tokens to Nextcloud's CSS variables using a **7-layer CSS architecture**
- Syncs Nextcloud's primary color, background color, and logo when you switch themes
- Provides optional toggles for hiding the login slogan and showing sidebar text labels
- Includes **344 Amsterdam Design System icons** for use across Nextcloud apps

## Who it's for

- **Nextcloud administrators** at Dutch government organizations who need to apply their organization's branding
- **Nextcloud app developers** who want their apps to look correct when NL Design is active
- **Design system teams** implementing NL Design System tokens in Nextcloud
